Decentralized Academic Integrity Evidence Platform

Novel Application

Turn paper submission, peer review, and citation records into tamper-resistant academic evidence.

The platform records content hashes, timestamps, review hashes, and citation declarations. The original paper is not stored on-chain, so private academic content stays off the public blockchain.

Integrity SHA-256 hash proves the manuscript has not changed.
Timestamping Submission time is recorded by local demo time or block time.
Non-repudiation Wallet signatures link actions to submitters.
Transparency Reviews and citation trails can be audited.

How MetaMask Connects

Install the MetaMask browser extension, open this page, and click Connect MetaMask. The browser asks for permission to expose your wallet address to this demo.

When the wallet is connected, paper, review, and citation submissions can trigger Sepolia transaction popups. The demo writes a small evidence message into transaction data.

Can It Review DOCX Papers?

Yes for the MVP workflow. Upload a .docx file, and the browser creates a SHA-256 file hash for blockchain evidence.

The demo does not read or store the full DOCX content on-chain. AI detection can be added later through GPTZero, Turnitin, or an OpenAI-based backend.
